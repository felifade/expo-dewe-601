Esquema eléctrico

Diagrama de conexión

Conecta los componentes según el siguiente esquema. Respeta la polaridad del buzzer.

1️⃣

Botón → Pin D2

Conecta un terminal del botón al pin digital D2 del Arduino. El otro terminal va a GND con resistencia pull-down de 10 kΩ.

2️⃣

Buzzer → Pin D8

Conecta el pin positivo (+) del buzzer al pin D8. El pin negativo (–) va directamente a GND del Arduino.

3️⃣

Alimentación

Conecta el Arduino por USB al PC (5 V). Verifica la polaridad antes de encender para evitar daños en el buzzer.

Programación

Código fuente

El siguiente código implementa la lógica del timbre con dos notas musicales.

timbre_completo.ino 
/*
 * Timbre con Arduino UNO
 * Pines: Botón → D2 | Buzzer → D8
 * Autor: Proyecto Timbre Arduino
 */

const int PIN_BOTON  = 2;
const int PIN_BUZZER = 8;

// Frecuencias en Hz
const int NOTA_A5 = 880;
const int NOTA_E5 = 659;

void setup() {
  pinMode(PIN_BOTON,  INPUT);
  pinMode(PIN_BUZZER, OUTPUT);
  Serial.begin(9600);
  Serial.println("Timbre listo ✓");
}

void loop() {
  int estado = digitalRead(PIN_BOTON);

  if (estado == HIGH) {
    Serial.println("¡Timbre pulsado!");
    tocarTimbre();
  }
}

void tocarTimbre() {
  tone(PIN_BUZZER, NOTA_A5, 200);
  delay(260);
  tone(PIN_BUZZER, NOTA_E5, 400);
  delay(500);
  noTone(PIN_BUZZER);
}

Proceso de desarrollo

FASE 01

Investigación y diseño

Selección de componentes, lectura de datasheets del Arduino UNO y buzzer piezoeléctrico. Diseño del esquema en Tinkercad.

FASE 02

Montaje en protoboard

Conexión física de todos los componentes según el diagrama. Verificación de continuidad con multímetro.

FASE 03

Programación y pruebas

Escritura del código en Arduino IDE. Pruebas de funcionamiento y ajuste de frecuencias del buzzer.

FASE 04

Documentación final

Elaboración de este sitio web, diagramas y manual de usuario para replicar el proyecto.
