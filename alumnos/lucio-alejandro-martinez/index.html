Timbre Inteligente con Arduino

Un sistema de timbre domótico desarrollado con Arduino UNO, sensor de presencia y alerta sonora personalizada. Conectividad simple, respuesta instantánea.

Resumen del proyecto

¿Qué es y cómo funciona?

El timbre Arduino es un dispositivo electrónico que detecta la pulsación de un botón y genera una señal sonora, integrando hardware y software en un sistema eficiente.

0 V Voltaje operación
0 Ω Resistencia buzzer
0 MHz Frecuencia Arduino
0 comp. Componentes clave
🎛️

Arduino UNO

Microcontrolador ATmega328P con 14 pines digitales. Procesa la señal del botón y controla el buzzer en tiempo real.


Controlador
🔘

Botón Pulsador

Interruptor momentáneo de 4 pines conectado a un pin digital con resistencia pull-down para lectura limpia.


Entrada
📢

Buzzer Piezoeléctrico

Genera el sonido de timbre mediante señal PWM. Frecuencia ajustable desde el código con la función tone().

// Timbre con Arduino UNO
const int pinBoton = 2;
const int pinBuzzer = 8;

void setup() {
  pinMode(pinBoton, INPUT);
  pinMode(pinBuzzer, OUTPUT);
}

void loop() {
  if (digitalRead(pinBoton) == HIGH) {
    tone(pinBuzzer, 880, 200);
    delay(250);
    tone(pinBuzzer, 660, 400);
    delay(500);
  }
}
