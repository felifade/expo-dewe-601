Pin 3 (Arduino)  ← Botón 2
Pin 4 (Arduino)  → Buzzer (+)
GND (Arduino)    → Buzzer (-) y Botones (GND)

Programación

Fase 4 · Semana 2–3

Desarrollo del Código

Se programó el Arduino con el IDE oficial. El código detecta cuando se presiona cualquiera de los dos botones y activa el buzzer con una duración configurable. 

// Timbre Automático - Código Principal
const int BTN1 = 2;
const int BTN2 = 3;
const int BUZZER = 8;

void setup() {
  pinMode(BTN1, INPUT);
  pinMode(BTN2, INPUT);
  pinMode(BUZZER, OUTPUT);
}

void loop() {
  if (digitalRead(BTN1) == HIGH || digitalRead(BTN2) == HIGH) {
    tone(BUZZER, 1000, 500);
    delay(1000);
  }
  delay(50);
}
