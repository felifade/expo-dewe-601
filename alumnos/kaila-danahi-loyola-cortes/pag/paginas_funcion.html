⚙️ Funcionamiento del Sistema

  • Se programan los horarios en Arduino.
  • El reloj interno controla el tiempo.
  • Cuando llega la hora establecida, el sistema activa el timbre.
  • Los horarios pueden modificarse fácilmente.
  • El ESP32 puede añadir conectividad WiFi para futuras mejoras.
