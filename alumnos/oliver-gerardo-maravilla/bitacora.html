📖 Bitácora del Proyecto

Descripción de la Bitácora

La bitácora del proyecto permite registrar todas las actividades realizadas durante el desarrollo del timbre inteligente con Arduino. Gracias a este seguimiento es posible documentar avances, identificar problemas y evaluar los resultados obtenidos en cada etapa.

Cronograma de Actividades

Fecha Actividad Descripción
01/06/2026 Investigación del proyecto Se investigó el funcionamiento de los timbres inteligentes y el uso de Arduino.
03/06/2026 Selección de materiales Se identificaron y reunieron los componentes necesarios para el proyecto.
05/06/2026 Diseño y armado del circuito Se realizaron las conexiones en la protoboard utilizando los componentes electrónicos.
07/06/2026 Programación de Arduino Se desarrolló el código para controlar el buzzer y el LED mediante un pulsador.
09/06/2026 Pruebas y corrección de errores Se verificó el funcionamiento general del sistema y se corrigieron fallas detectadas.
10/06/2026 Documentación del proyecto Se elaboró la página web y la documentación técnica del proyecto.

Problemas Encontrados

Durante el desarrollo del proyecto se presentaron algunos inconvenientes, como conexiones incorrectas en la protoboard, errores en la programación y dificultades para sincronizar algunos componentes electrónicos.

Estos problemas fueron solucionados mediante pruebas continuas, revisión del código y trabajo colaborativo entre los integrantes del equipo.

Aprendizajes Obtenidos

El proyecto permitió fortalecer conocimientos relacionados con la electrónica, la programación y el diseño de sistemas automatizados. Además, se desarrollaron habilidades de investigación, organización y trabajo en equipo.

Conclusiones de la Bitácora

La elaboración de la bitácora permitió llevar un control detallado del progreso del proyecto, facilitando la identificación de errores y la planificación de actividades futuras.

Gracias a este seguimiento fue posible completar exitosamente el timbre inteligente y documentar cada una de las etapas de su desarrollo.
