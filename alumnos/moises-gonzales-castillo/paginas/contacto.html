FAQ

Preguntas Frecuentes

🤔

¿El sensor mide SpO₂?

Sí. El módulo MAX30102 puede medir tanto la frecuencia cardíaca como la saturación de oxígeno en sangre (SpO₂) simultáneamente.

📱

¿Funciona en móvil?

Completamente. La interfaz es 100% responsive y funciona en smartphones, tablets y computadoras de escritorio.

🔋

¿Consume mucha batería?

El sensor MAX30102 está optimizado para bajo consumo energético, ideal para dispositivos portátiles con batería limitada.

💻

¿Puedo ver el código fuente?

El proyecto está desarrollado con HTML, CSS y JavaScript planos, sin frameworks. Puedes inspeccionarlo directamente en el navegador.
