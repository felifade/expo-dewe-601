// 03 — Prototipo físico

Componentes de hardware

🖥️

ESP32

Microcontrolador de doble núcleo con WiFi y Bluetooth integrado. El cerebro del timbre.

CPUXtensa LX6 240MHz
RAM520 KB SRAM
WiFi802.11 b/g/n
GPIO34 pines
⚡

Protoboard

Placa de prototipado sin soldadura para conexión rápida durante el desarrollo.

Puntos830 contactos
Buses+ / −
Voltaje3.3V / 5V
TipoPrototipado rápido
🔴

Botón push

Botón normalmente abierto. Accionado por la visita para activar el sistema.

TipoN.O. momentáneo
Pull-upInterno ESP32
GPIOPin configurable
DebounceSoftware
🔌

Cables macho-macho

Jumpers para conectar ESP32, protoboard y periféricos de forma ordenada.

TipoMacho-Macho
Longitud10 – 20 cm
ColoresCódigo estándar
MaterialCobre estañado
📷

Módulo cámara

OV2640 compatible con ESP32-CAM para transmisión de video en tiempo real.

Resolución2 Megapíxeles
Lente160° gran angular
StreamMJPEG / JPEG
InterfazDVP 8-bit
🔊

Módulo de audio

Altavoz o DFPlayer Mini para reproducir el mensaje de bienvenida automático.

TipoDFPlayer / buzzer
Voltaje3.3V – 5V
ControlPWM / Serial
FormatoMP3 / WAV
