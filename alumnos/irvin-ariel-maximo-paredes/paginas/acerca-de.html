MISIÓN

Tecnología accesible para el hogar inteligente

SmartBell nació como un proyecto académico con la convicción de que la domótica y el IoT no tienen que ser costosos ni complicados. Usando componentes de bajo costo como el ESP32 y una protoboard, demostramos que cualquier persona puede construir su propio sistema de seguridad inteligente.

Lo que nos guía

🔓
Código abierto
Todo el código del proyecto está disponible públicamente para que otros puedan aprender, mejorar y reutilizarlo en sus propios proyectos IoT.

Accesibilidad

Privacidad

Este proyecto es también una herramienta educativa: documenta cada decisión técnica para que otros estudiantes puedan replicarlo y mejorarlo.
