FAQ

Preguntas frecuentes

¿Puedo replicar este proyecto en casa? +
Sí, todo el código está disponible en nuestro repositorio. Necesitarás un ESP32, una protoboard, un botón pulsador y cables jumper. El costo total del hardware es de aproximadamente $5–10 USD.
¿Funciona sin conexión a internet? +
En modo local (red LAN) sí, siempre que el teléfono y el ESP32 estén en la misma red WiFi. Para notificaciones push remotas necesitas un broker MQTT con acceso a internet.
¿Qué cámara usa el sistema? +
El prototipo actual usa la cámara del navegador web (MediaDevices API) como simulación. Para una implementación real se recomienda el módulo ESP32-CAM que integra cámara OV2640 directamente en el microcontrolador.
¿El proyecto está en desarrollo activo? +
Sí, actualmente estamos trabajando en la integración con ESP32-CAM para video real, detección de movimiento por PIR y una app móvil nativa con React Native.
