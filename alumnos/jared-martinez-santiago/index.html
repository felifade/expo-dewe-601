📋 Introducción

¿De qué trata el proyecto?

Una solución tecnológica accesible para medir el ritmo cardíaco usando electrónica de bajo costo.

🔵

¿Qué es Arduino?

Arduino es una plataforma de electrónica de código abierto que combina una placa de microcontrolador y un entorno de desarrollo. Permite crear prototipos electrónicos interactivos de forma rápida y económica, ideal para proyectos educativos y de innovación.

❤️

Sensor MAX30102

El MAX30102 es un sensor de oximetría de pulso y frecuencia cardíaca de alta integración. Utiliza LEDs rojo e infrarrojo para medir la variación de luz absorbida por la sangre, calculando con precisión el ritmo cardíaco y la saturación de oxígeno (SpO₂).

🎯

Objetivo del Proyecto

Construir un monitor de ritmo cardíaco funcional usando Arduino y el sensor MAX30102 para demostrar cómo la tecnología accesible puede aplicarse en el monitoreo de salud, abriendo posibilidades de innovación en el área biomédica.

🌍

Importancia del Proyecto

El acceso a tecnología de monitoreo cardíaco básico puede ser transformador en comunidades con recursos limitados. Este proyecto demuestra que con menos de 300 pesos en componentes, es posible construir un dispositivo que en hospitales cuesta miles de pesos.
