Componentes del Sistema

Arduino Uno

Microcontrolador principal encargado de ejecutar la programación y controlar todos los dispositivos del sistema.

Pantalla LCD 16x2

Muestra la fecha, hora y el estado actual del sistema de timbre escolar programado.

Pulsador (Botón)

Permite realizar acciones manuales como configuración, pruebas o ajustes del sistema.

Resistencia

Se utiliza para proteger los componentes electrónicos y garantizar el correcto funcionamiento del pulsador.

Buzzer

Genera el sonido del timbre cuando la hora actual coincide con un horario programado.

Caja de Texto Programada

Elemento implementado mediante código para mostrar mensajes informativos, horarios y estados del sistema dentro de la interfaz.

Cableado de Conexión

Permite la comunicación entre el Arduino, la pantalla LCD, el buzzer y el pulsador.

Sistemas del Timbre

¿Cómo Funciona el Sistema?

El sistema de timbre escolar programado funciona mediante la lectura del tiempo en tiempo real. Cuando la hora actual coincide con un horario establecido, se activa automáticamente un buzzer o campana que indica el cambio de clase, el recreo o el inicio y finalización de actividades escolares.

Gracias a la automatización, se evita la activación manual del timbre, mejorando la puntualidad y organización dentro de la institución.

Horarios del Timbre

  • 07:00 AM → Entrada
  • 08:00 AM → Primera Clase
  • 10:00 AM → Recreo
  • 10:30 AM → Fin del Recreo
  • 01:00 PM → Salida

Bitácora del Proyecto

Durante el desarrollo del Timbre Escolar Programado se realizaron diversas actividades para garantizar el correcto funcionamiento del sistema.

  • Montaje del circuito utilizando Arduino Uno.
  • Conexión y configuración de la pantalla LCD 16x2.
  • Instalación del buzzer para la emisión del timbre escolar.
  • Implementación del pulsador y resistencia para el control del sistema.
  • Programación de la fecha y hora mostradas en la pantalla.
  • Desarrollo de una caja de texto programada para mostrar mensajes y estados del sistema.
  • Realización de pruebas de funcionamiento y corrección de errores.
  • Validación final de los horarios automáticos del timbre escolar.

Como resultado, se obtuvo un sistema capaz de mostrar la fecha y hora en tiempo real y activar automáticamente el timbre de acuerdo con los horarios establecidos.
